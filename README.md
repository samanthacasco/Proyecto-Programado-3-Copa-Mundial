## Descripción

Este proyecto consiste en un simulador de mundial de fútbol desarrollado en **Python** utilizando programación orientada a objetos e interfaz gráfica con **Tkinter**.

El sistema permite administrar países, selecciones, entrenadores y jugadores, organizar un mundial, simular todos los partidos de la fase de grupos y las fases eliminatorias, generar estadísticas y producir reportes del torneo.

---

## Funcionalidades

El sistema permite:

* Registrar países.
* Registrar selecciones nacionales.
* Registrar futbolistas.
* Registrar entrenadores.
* Configurar el mundial creando grupos.
* Simular la fase de grupos.
* Simular las fases eliminatorias hasta obtener un campeón.
* Consultar estadísticas del torneo.
* Visualizar:

  * Ranking de goleadores.
  * Tarjetas por selección.
  * Tabla general de posiciones.


4. Registrar la información necesaria.
